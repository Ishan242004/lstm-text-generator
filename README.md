# LSTM Text Generator

This project implements a text generation model using LSTM (Long Short-Term Memory).

## Dataset
Shakespeare Sonnets dataset from Kaggle

## Features
- Data preprocessing
- Tokenization
- Sequence generation
- LSTM model training
- Text generation

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy

## Output
The model generates Shakespeare-like text based on seed input.

## Example
Input: "shall i compare thee"
Output: "shall i compare thee to a summer day..."
